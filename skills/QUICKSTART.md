# 🚀 快速开始指南

## 一键安装

```bash
# 1. 运行安装脚本
cd skills/memory-analyzer
./install.sh

# 2. 配置 API 密钥（如果安装时未配置）
echo 'GEMINI_API_KEY=your_api_key' > ../../.env
```

## 三种使用方式

### 方式 1️⃣: 基础分析（AI 自动识别主角）

```bash
./run.sh ~/Photos/2025
```

**适合**: 照片大多是独照或主角明显

---

### 方式 2️⃣: 提供主角参考（推荐⭐）

```bash
# 先提取主角特征
python3 ../../extract_protagonist.py ~/selfie.jpg

# 然后运行分析
./run.sh ~/Photos/2025
```

**适合**:
- 照片混杂
- 主角不常入镜
- 追求最高准确度

**准确度**: 95% vs 70%

---

### 方式 3️⃣: 调整分析后重跑

```bash
# 编辑 Phase 2 prompt
# vim ../../memory_profiler.py

# 仅运行 Phase 2（使用缓存）
./run.sh ~/Photos/2025 --phase2-only
```

**适合**:
- 第一次分析结果不满意
- 想尝试不同的分析角度
- 需要反复调整

**时间**: 1-2 分钟 vs 15-25 分钟

## 常见问题

### Q: 如何查看分析质量？

```bash
python3 ../../quality_monitor.py
```

### Q: 支持哪些照片格式？

✅ JPG, JPEG, PNG, HEIC, WebP
✅ MPO（自动转换）

### Q: 分析失败怎么办？

```bash
# 查看错误日志
cat ../../memory_profiler.log | grep ERROR
```

### Q: 有多少张照片比较合适？

📸 **推荐**: 500-2000 张
📅 **跨度**: 至少 6 个月到 1 年
